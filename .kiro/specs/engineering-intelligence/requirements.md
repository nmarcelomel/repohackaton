# Feature: Plataforma Engineering Intelligence y Gestión del Cambio

## Descripción
Plataforma para Seguros Bolívar que consolida métricas de ingeniería (DORA), gestión del talento, bienestar y valor de negocio.

## Requisitos Funcionales
- RF-01: Layout con Sidebar (4 rutas) y Topbar (notificaciones)
- RF-02: Módulo Bienestar (/bienestar) - placeholder
- RF-03: Módulo Talento (/talento) - placeholder
- RF-04: Módulo Flujo (/flujo) - placeholder
- RF-05: Módulo Negocio (/negocio) - placeholder
- RF-06: Mock data centralizado (teams, users, initiatives, kudos)

## Requisitos No Funcionales
- React 18.3 + Vite 5.3 + TypeScript 5.4 strict
- Tailwind CSS 3.4 con paleta corporativa
- React Router 6.23

## Criterios de Aceptación
1. npm run build sin errores ✅
2. 4 rutas navegables desde sidebar
3. mock-data.json válido y tipado
